## Context-Aware Variations

The catalogue panel adapts based on which studio is active:

### In Data Science Studio
- Emphasizes: ML models, data tools, notebooks
- Featured categories: PyTorch, TensorFlow, scikit-learn
- Quick actions: Import model, Load dataset

### In Low-Code Studio
- Emphasizes: Workflow nodes, integrations, connectors
- Featured categories: Triggers, Actions, Logic nodes
- Quick actions: Add node, Load template

### In Declarative Builder
- Emphasizes: Agent templates, prompt libraries, guardrails
- Featured categories: Pre-built agents, Memory systems
- Quick actions: Load template, Clone agent

### In Business Analytics
- Emphasizes: Visualization types, data sources, AI insights
- Featured categories: Chart types, Connectors, ML models
- Quick actions: Add chart, Connect data

---

## User Interactions

### 1. Browsing
- **Expand Category**: Click to see items within category
- **Search**: Real-time filtering across all items
- **Filter**: By category, type, price, popularity
- **Sort**: Popular, Recent, A-Z, Price

### 2. Item Actions
- **View Details**: Click item or "View" button
- **Quick Add**: Add directly to current project
- **Add to Favorites**: Star for quick access
- **Share**: Share with team members

### 3. Management
- **Favorites**: Quick access to frequently used items
- **Custom Items**: View/edit user-created tools/agents
- **Import**: Upload custom tools or agents
- **Refresh**: Update catalogue with latest items

---

## Integration Points

### Adding Item to Project
When user clicks "Add to Project" from detail modal:

1. **Data Science Studio**: Imports package, adds to requirements
2. **Low-Code Studio**: Adds node to available palette
3. **Declarative Builder**: Pre-fills configuration form
4. **Business Analytics**: Adds data source or visualization

### Quick Add Behavior
Direct "Add" button in search results or grid:
- Minimal confirmation
- Adds with default configuration
- Notification: "OpenAI GPT-4 added to project"

---

## Design Notes

- **Always Accessible**: Panel always visible in right sidebar
- **Context-Aware**: Content prioritized by current studio
- **Search-First**: Powerful search for quick discovery
- **Rich Metadata**: Ratings, usage stats, pricing visible
- **Quick Preview**: Hover cards for rapid browsing
- **Drag & Drop**: Drag items directly into workspace
- **Favorites**: Personalized quick access
- **Community**: User ratings, reviews, shared items
- **Versioning**: Track different versions of tools/agents
- **Dependencies**: Auto-install required dependencies
- **Documentation**: Inline docs, examples, tutorials
- **Performance**: Lazy loading, pagination for large catalogues
